# 综合体育数据辅助分析系统 - 前端开源参考版本（Python + uniapp架构）

## 📖 项目简介
本项目是一款专为**体育赛事数据辅助分析、实体店辅助出票记录展示**设计的高级全端应用系统，涉及竞彩足球、竞彩篮球、北京单场、排列3、排列5、任选九、足球14场、七星彩、大乐透、福彩3D、快乐8、七乐彩、双色球等各种常见数据分析及历史走势记录（**本项目代码仅涉及接口开发、赛事分析走势，前端展示图表等，不包含任何打票核心算法及投注业务**）。

当前开源库采用 `Uni-App` (适用于微信小程序、H5、Android/iOS客户端) 结合少部分纯前端示例代码呈现。
